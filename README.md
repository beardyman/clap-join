#Clap

## Usage
```shell script
clap "Why did you do this?"
# Output: 👏Why👏did👏you👏do👏this?👏
```

### Use it in your code too!!! 
```js
const clap = require('clap');

console.log(clap('Why did you do this?'))
// Output: 👏Why👏did👏you👏do👏this?👏
```

## Installation
For CLI
```shell script
npm i -g clap-join
```
For code
```shell script
npm i clap-join
```
